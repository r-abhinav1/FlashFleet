# FlashFleet

FlashFleet is a full-stack productivity and learning assistant that integrates a Pomodoro Timer, Flashcard creation, and AI-generated Quizzes. It is designed to support focused study, personalized revision, and efficient practice, offering students an all-in-one productivity environment.
